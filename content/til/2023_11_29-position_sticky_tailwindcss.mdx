We can use `sticky` class when using tailwind to make an element fixed on screen while the parent element is visible.

Code:
```html
<div class="relative">
  <h1 class="sticky top-0">Element</h1>
</div>
```

CSS Equivalent:
```css
.parent-element {
  position: relative;
}
.element {
  position: sticky;
  top: 0;
}
```
